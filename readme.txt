CPW-Custom Post Widget adds custom widget to widget area where you can customize posts shown to sidebar
== Description ==

CPW-Custom Post Widget Type is the perfect plugin to create Custom Posts Widget to show in your sidebar in a user friendly way, just like managing your regular widgets, it enable you to customize your posts colors and more.

The plugin implements the following functions, from which you can build your own custom UI and display code:

== Installation ==
1.Upload 'CPW-Custom Post Widget' to the '/wp-content/plugins/' directory.
2.Activate the plugin through the 'Plugins' menu in WordPress.
3.Click the Appreance > Widgets and add CPW Widget at your place.
4.Customize your posts display with bunch of options.
